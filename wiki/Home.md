# Welcome to the ChoreLoop Wiki!

ChoreLoop is a shared household task-management system built on a **Flat Trust Model**. It replaces restrictive permissions with **Soft Accountability**, using high visibility and activity logging to encourage responsible coordination within a single household.

## 🛠 Project Overview

- **Backend:** PHP (RESTful API)
- **Android:** Kotlin + Jetpack Compose
- **Web:** Vanilla JS + ES6 Modules
- **Database:** MySQL

## 🎯 Vision

ChoreLoop aims to eliminate household friction by providing a transparent, zero-barrier platform for task coordination. No passwords, no roles—just chores.
